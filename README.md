关于零点远控
因为工作需要出于以上两点，所以业余时间开发了这款软件
通过公司自己架设服务器 30人的客服团队一个月也就90块左右的带宽
软件服务器基本不占系统资源(10MB内存,cpu 一核一G 基本不会超过10%)
只是带宽相对较高，可根据具体使用情况选择服务器带宽
通常每远程一小时的成本在0.2元以下(高清模式)


零点经过了一年多公司客服部的使用现在基本已经稳定完善
并且我也将持续优化

零点目前采用端到端加密，强度达到金融级。
开源出来也正是向大家证明零点是非常安全的
即使你拿着源码也无法连接别人的终端
